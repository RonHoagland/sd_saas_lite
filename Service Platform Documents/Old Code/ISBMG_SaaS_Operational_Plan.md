# 2. Product Tier Structure

## Lite -- \$24 per seat

-   Max 10 seats
-   3GB storage
-   Manual workflow
-   No automation
-   No SMS
-   No system email sending
-   Catalog-only inventory
-   CSV export only

## Plus -- \$49 per seat

-   Unlimited seats
-   Automation
-   SMS (points system)
-   System email sending (limited)
-   Expanded storage
-   Optional paid storage add-ons

## Pro -- \$98 per seat

-   Built on Plus
-   Advanced automation
-   Dispatch board
-   Inventory movement
-   Vendor + PO system
-   Advanced reporting
-   Multi-location
-   API access

## Enterprise -- \$145 per seat (Future Tier)

-   Enhanced scaling tools
-   Extended automation rules
-   Advanced reporting dashboards
-   Priority support

# 8. Sales Tax Compliance

## Minnesota

-   SaaS taxable in MN
-   Must register with MN Department of Revenue
-   File monthly or quarterly depending on volume

## Other States

-   Economic nexus generally triggered at \$100,000 revenue or 200
    transactions
-   Monitor thresholds annually

## Recommended Tool

Stripe Tax (\~0.5% of taxable revenue)
